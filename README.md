# DeFi Yield Tracker (Web3)

This project helps you track DeFi deposits and estimated yield using simple blockchain logic.

It is made for beginners who want to learn how DeFi apps work behind the scenes.

## What this project does

- Stores user token deposits
- Simulates yield calculation
- Shows total balance and profit
- Uses simple blockchain-style data flow

## Why this project is useful

You will learn:
- How DeFi platforms track balances
- How yield is calculated
- How to structure a Web3 backend
- How to organize crypto project files

## Tech used

- JavaScript
- Node.js
- Blockchain logic (simulated)
- DeFi concepts

## How to run

1. Install Node.js  
2. Clone this repo  
3. Open terminal in project folder  
4. Run: `node app.js`  

## Files overview

- `app.js` → Main app file  
- `config.js` → Settings  
- `deposit.js` → Deposit logic  
- `yield.js` → Yield calculator  
- `user.js` → User wallet logic  
- `pool.js` → DeFi pool logic  
- `utils.js` → Helper functions  
- `logger.js` → Console logs  
- `sample.env` → API config  
- `package.json` → Project info

## Notes

This project is for learning only.  
Do not use real wallets or funds.
